PSU Tree Carbon Database

ฐานข้อมูลต้นไม้และการกักเก็บคาร์บอนภายในมหาวิทยาลัยสงขลานครินทร์ วิทยาเขตภูเก็ต

ต้นไม้ที่มีความโดดเด่นในพื้นที่ศึกษา ซึ่งมีบทบาทสำคัญในการดูดซับคาร์บอนและสร้างความร่มรื่นให้กับพื้นที่

ศรีตรัง (Jacaranda)

โดดเด่นด้วยดอกสีม่วงสวยงามที่บานสะพรั่งทั่วพื้นที่ เป็นสัญลักษณ์แห่งความภาคภูมิใจและภาพจำที่โดดเด่นของมหาวิทยาลัย

จามจุรี (Rain Tree)

ต้นไม้ขนาดใหญ่ที่มีเส้นผ่านศูนย์กลางลำต้นมากที่สุดในพื้นที่ศึกษา ให้ร่มเงากว้างขวางและมีบทบาทสำคัญในการดูดซับคาร์บอน

ทุเรียน (Durian)

ต้นไม้ที่มีเพียงต้นเดียวในพื้นที่มหาวิทยาลัย มีความโดดเด่นเฉพาะตัว และเป็นพืชเศรษฐกิจที่สำคัญของประเทศไทย

ประดู่ (Burmese Padauk)

ต้นไม้ประจำจังหวัดภูเก็ต มีเนื้อไม้แข็งแรงทนทาน และมีคุณค่าทางเศรษฐกิจและสิ่งแวดล้อม

จำนวนต้นไม้ที่ศึกษา

175 ต้น

ชนิดพันธุ์ต้นไม้

43 ชนิด

ปริมาณคาร์บอน

42,829.63 kgC

ปริมาณก๊าซเรือนกระจกที่กักเก็บได้

157,042.26 kgCO₂e

PSU Tree Carbon Database

Web GIS system for tree inventory and carbon stock analysis in PSU Phuket Campus

🌿 Project Purpose

โครงการนี้จัดทำขึ้นเพื่อเป็นฐานข้อมูลต้นไม้ภายในมหาวิทยาลัย โดยมีวัตถุประสงค์หลักในการรวบรวมข้อมูลต้นไม้ วิเคราะห์ปริมาณคาร์บอนที่ถูกกักเก็บ และแสดงผลข้อมูลผ่านระบบ Web GIS เพื่อเป็นเครื่องมือสนับสนุนการอนุรักษ์สิ่งแวดล้อม
