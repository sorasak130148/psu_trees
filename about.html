🌿 Project Overview

โครงการ PSU Tree Carbon Database เป็นระบบฐานข้อมูลเชิงพื้นที่ (Spatial Database) ที่พัฒนาขึ้นเพื่อจัดเก็บข้อมูลต้นไม้ภายในมหาวิทยาลัยสงขลานครินทร์ วิทยาเขตภูเก็ต โดยมุ่งเน้นการวิเคราะห์การกักเก็บคาร์บอนของต้นไม้และการใช้ข้อมูลเชิงพื้นที่ผ่าน Web GIS
